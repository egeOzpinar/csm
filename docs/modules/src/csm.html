csm¶

Usage

use csm;

Chapel Statistics Module is a statistics module for Chapel which has statistics functions. It provides fundamental statistics functions. Forms of the functions except pdf() function:

proc funcName(var args) {}

or

proc funcName(array) {}

Examples¶

Basic Usage¶

For variadic arguments:

// example1.chpl
use csm;

var x = mean(12,32,43,2,5,743,54);

For arrays:

// example2.chpl
use csm;

var array = [12,32,43,2,5,743,54];
var m = mean(array);

Compile the code samples at the above:

chpl example1.chpl

and

chpl example2.chpl

return same result.
